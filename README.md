hue program language
====================
##Usage:
  `python hue.py yourhue.hue`
####or as module:
```
import hue
hue.run(yourcode)
```

##commands reference
|   hue    | brainfuck |
|----------|-----------|
| h        | >         |
| hu       | <         |
| hue      | +         |
| hueh     | -         |
| huehu    | .         |
| huehue   | ,         |
| huehueh  | [         |
| huehuehu | ]         |
